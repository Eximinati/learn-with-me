# Learn With Me

A planning and study-organization app for learners using structured study plans and book uploads, built with **Next.js** and **Supabase**.

## Goals
- Make complex study plans easier to understand through structured sections, milestones, and progress tracking.
- Keep all required books, notes, and references organized alongside each plan step.
- Provide a clear, searchable interface for navigating long learning paths.

## Tech stack
- **Next.js** (App Router)
- **Supabase** (Auth, Postgres, Storage)
